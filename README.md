# JDir

Windows 전용 TUI(Terminal UI) 파일 탐색기. [Textual](https://github.com/Textualize/textual) 기반으로 제작되었으며, 터미널에서 키보드만으로 파일을 탐색하고 관리할 수 있다.

## 요구사항

- Windows 10/11
- Python 3.10 이상
- [Textual](https://github.com/Textualize/textual)

```
pip install textual
```

## 실행

```
python jdir.py
```

실행 시 현재 작업 디렉터리(또는 저장된 시작 폴더)에서 탐색을 시작한다.


### 항목 표시 형식

| 접두사 | 의미 |
|--------|------|
| `^` | 상위 폴더 (`..`) |
| `[D]` | 드라이브 (내 PC 뷰) |
| `>  [폴더명]` | 폴더 |
| `[!]` | 실행 파일 (`.exe` `.bat` `.cmd` `.msi` `.com` `.ps1` `.py`) |
| `[-]` | 문서 파일 (`.doc` `.docx` `.ppt` `.pptx` `.txt` `.csv` `.md`) |

목록 정렬 순서: 폴더 → 실행 파일 → 문서 파일 (각 그룹 내 이름 오름차순)

## 키보드 단축키

### 탐색

| 키 | 동작 |
|----|------|
| `↑` / `↓` | 커서 이동 |
| `Enter` | 폴더 진입 / 파일 열기 |
| `→` | 폴더 진입 |
| `←` | 커서가 `..`에 있으면 상위 폴더로 이동, 아니면 커서를 목록 맨 위로 이동 |
| `Alt+←` | 탐색 히스토리 뒤로 |
| `Alt+→` | 탐색 히스토리 앞으로 |
| `Tab` | 상단 바 요소 간 포커스 순환 (목록 → Claude 버튼 → 경로 입력 → 이동 버튼) |
| `Ctrl+R` | 경로 입력창으로 포커스 이동 |

### 선택

| 키 | 동작 |
|----|------|
| `Space` | 커서 항목 선택/해제 토글 (커서 이동 없음) |
| `Shift+↓` | 커서를 유지한 채 범위 선택 확장 (아래 방향) |
| `Shift+↑` | 커서를 유지한 채 범위 선택 확장 (위 방향) |
| `Ctrl+A` | 현재 폴더 내 모든 항목 선택 |

**범위 선택 동작 방식**
- `Shift+↓/↑`를 처음 누른 위치가 앵커. 앵커 ~ 현재 커서 범위의 항목이 토글됨
- 반대 방향으로 되돌아가면 범위가 줄어들며 해제됨 (고무줄 방식)
- 일반 `↑`/`↓` 또는 다른 키를 누르면 세션 종료
- `Space`와 `Shift+방향키`를 혼용해 비연속 다중 선택 가능

### 파일 작업

| 키 | 동작 |
|----|------|
| `Ctrl+C` | 선택 항목(또는 커서 항목) 복사 |
| `Ctrl+X` | 선택 항목(또는 커서 항목) 잘라내기 |
| `Ctrl+P` | 현재 폴더에 붙여넣기 |
| `Ctrl+D` | 선택 항목(또는 커서 항목) 삭제 (확인창 표시) |
| `Ctrl+Z` | 마지막 붙여넣기 취소 (복사→파일 삭제, 이동→원위치 복원) |

> 선택된 항목이 없으면 커서가 위치한 항목에 작업이 적용된다.

### 기타

| 키 | 동작 |
|----|------|
| `ESC` | 선택 항목이 있으면 전체 해제 + 클립보드 초기화. 아무것도 없으면 종료 확인 |

## 주요 기능 상세

### 탐색 히스토리 (v0.4)

방문한 폴더 경로를 포인터 방식으로 기록한다.

- `Alt+←` / `Alt+→`는 포인터만 이동하며 히스토리를 새로 쌓지 않는다
- 직접 폴더를 탐색하면 현재 포인터 이후의 히스토리가 삭제되고 새 항목이 추가된다
- `Ctrl+Z`로 인한 자동 폴더 이동도 히스토리에 기록된다

### 실행취소 (v0.4)

붙여넣기 작업만 취소 가능하다.

- **복사 후 붙여넣기 취소**: 생성된 파일/폴더를 삭제
- **이동(잘라내기+붙여넣기) 취소**: 파일/폴더를 원래 위치로 복원
- 취소 후 해당 작업이 수행된 폴더로 자동 이동

> 삭제(`Ctrl+D`)는 실행 전 확인창을 통해 보호되므로 별도의 취소 기능을 제공하지 않는다.

### 클립보드 카운트다운 (v0.3)

복사 후 붙여넣기를 완료하면 클립보드 내용이 10초 후 자동 삭제된다. 상태 바에 남은 시간이 표시된다.

### 시작 폴더 설정

경로 입력창에 폴더 경로를 입력하고 `이동` 버튼 또는 `Enter`를 누르면 해당 폴더로 이동하며 설정이 저장된다. 다음 실행 시 저장된 폴더에서 시작한다. 경로를 비우고 이동하면 설정이 초기화된다.

설정 파일 위치: `~/.claude/jdir_config.json`

### Claude 실행

`Claude 실행` 버튼을 누르면 현재 탐색 중인 폴더를 작업 디렉터리로 하여 `claude` CLI가 실행된다.

## 파일 분류 기준

```python
EXEC_EXTENSIONS = {'.exe', '.bat', '.cmd', '.msi', '.com', '.ps1', '.py'}
DOC_EXTENSIONS  = {'.doc', '.docx', '.ppt', '.pptx', '.txt', '.csv', '.md'}
```

위 확장자에 해당하지 않는 파일은 현재 목록에 표시되지 않는다.
